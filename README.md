## 🌙 Sobre o projeto

O **Mimo de Aniversário** foi desenvolvido como uma experiência digital personalizada para clientes da marca **Thau Luna Acessórios**.

O site funciona como um presente interativo:
- Envelope animado
- Mensagem personalizada
- Cupom surpresa
- Confetes
- Foto do cliente
- Experiência delicada e emocional

Tudo pensado para transformar um simples desconto em um momento especial. 💕
